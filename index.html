SASS Essential Mixins

Set of useful SASS/SCSS mixins for lazy front end developers.

Mixin list

  • borderRadius
  • selection
  • linearGradient

borderRadius ($size...)

Sets "border-radius" property. Parameters are passed in standard CSS order (top-left, top-right, bottom-right, bottom-left). Omited values will be replaced by 0.

+borderRadius(16px) @include borderRadius(16px);
+borderRadius(16px, 24px, 16px, 4px) @include borderRadius(16px, 24px, 16px, 4px);
+borderRadius(24px, 24px) @include borderRadius(24px, 24px);

selection selection($type)

Sets "user-select" property. Has "noSelection" child mixin which passes "none" value property as default.

Only text could be selected, no nodes.
+selection(text) @include selection(text);
You can't select me!
+noSelection @include noSelection;
